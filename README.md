# vite-react-typescript-starter

## 説明

PixPal Editorは、ブラウザ上でピクセルアートを作成・編集できるシンプルなエディタです。

## 使用技術

- React
- TypeScript
- Vite
- Tailwind CSS
- @uiw/react-color
- lucide-react

## インストール方法

```bash
npm install
```

## 実行方法

```bash
npm run dev
```

## 主要な機能

- ピクセルアートの作成・編集
- テンプレートの選択 (default-man, love)
- 色の選択・編集
- 作品のダウンロード・インポート

## テンプレート

- default-man: デフォルトの男性のテンプレート
- love: ハートのテンプレート
